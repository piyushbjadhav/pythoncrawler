What is ADEA PASS?

The ADEA Postdoctoral Application Support Service (PASS) is a centralized application service for advanced dental education programs in dentistry. Save time and energy by completing one application for multiple programs. While advanced dental education programs accept the ADEA PASS applications, they all have different requirements, deadlines and admissions timelines.  

 

The ADEA PASS Search Engine allows applicants to view this information and more for each participating program. Applicants may also search for programs by name, program type and/or state. The program information is updated directly by the programs. Therefore, programs should be contacted directly with questions regarding the posted information. To access the ADEA PASS Search Engine, visit programpages.passweb.org/search.

 

ADEA PASS is open from May through February each year. Applicants can create only one application per cycle.

 

It is the applicant’s responsibility to read, understand and follow all ADEA PASS and program-specific instructions.
