About PASS

The Pioneer Administrative Software System (PASS) is used for course registration, payment, financial aid, administrative functions, and more. Log in to PASS.

About PASS Express

PASS Express provides a streamlined interface to PASS, especially for mobile devices.   PASS Express includes some key features for students, faculty and advisors.  Discover what’s there for you. Log in to PASS Express.
