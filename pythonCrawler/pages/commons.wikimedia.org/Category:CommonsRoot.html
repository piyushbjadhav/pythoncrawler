This is the Root Category, the top level node in Commons tree data structure from which every other category is accessible.

  • The main category for contents on Commons is Category:Topics
  • The main category for our tree data structure is Category:Categories
  • The main category for Commons-related administration and maintenance files is Category:Commons.

See Commons:Categories for an introduction on using categories on Commons and other Wikimedia projects.
